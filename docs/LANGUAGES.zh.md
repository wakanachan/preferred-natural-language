## 语言代码格式

所有语言代码都遵循 [BCP-47](https://tools.ietf.org/html/bcp47) 标准格式：

```
language-code[-script-code][-country-code]
```

### 示例

- `en-US` - 美国的英语
- `zh-CN` - 中国的中文（简体）
- `ja-JP` - 日本的日语
- `pt-BR` - 巴西的葡萄牙语

## 添加新语言

要添加新语言支持：

1. 将语言代码和名称添加到 `src/languageNames.ts`
2. 在 `__tests__/unit/languageNames.test.ts` 中添加适当的测试用例
3. 运行测试确保一切正常工作
4. 提交拉取请求

## 语言检测优先级

工具按以下顺序检测语言偏好：

1. **配置文件** (`.preferred-language.json`)
2. **自定义环境变量** (`*_NATURAL_LANGUAGE`)
3. **操作系统语言设置**
4. **标准环境变量** (`LANGUAGE`, `LC_*`, `LANG`)
5. **HTTP Accept-Language 头**
6. **回退选项** (`en-US`)

更多详情请查看[主 README](../README.zh.md)。
