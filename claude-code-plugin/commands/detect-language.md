---
description: Detect the current system's preferred natural language setting
---

Please use the MCP tool `detect-language` to detect the user's preferred natural language and display the result in a user-friendly format.

The tool will return:
- Language code (e.g., zh-CN, en-US, ja-JP)
- Language name (e.g., 简体中文, English)
- Detection source (configuration file, OS locale, environment variable, etc.)
- Confidence level (high, medium, low)
